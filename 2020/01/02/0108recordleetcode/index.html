defaultdict

from collections import defaultdict
lookup = defaultdict(int)

默认lookup[‘xxx’] = 0

还可以 lookup = defaultdict(list)

默认lookup[‘xxx] = []比直接lookup = {}，可以预防keyerror

/ 和 //

“ / ” 为浮点数除法，返回浮点结果

“ // ” 表示整数除法，返回不大于结果的一个最大整数

枚举

int romanCharToInt(char c)
{
    int r = -1;
    switch (c)
    {
        case 'I':
            r = 1;
            break;
    }   
    return r;

}

字符串string

string c = "abcdefg";
c.find("b");  //返回1，返回b的下标，find即在字符串c中查找对应字符串的下标位置。
c = c.substr(0,c.length()-1); //substr，args为下标初始和结束位置。

二维数组定义方式

如设计到动态规划时，

  1. 用vector表示二维数组：
    表示m行，n列，所有的数为0。

vector<vector> dp(m, vector(n,0));

  1. 普通的定义方式
    int dp[m][n];
    在线笔试不会报错，但是VS可能会报错

    std::vector<std::vector > vec(row,vector(col,0));

    int size_row = vec.size(); //获取行数

    int size_col = vec[0].size(); //获取列数

  1. vector ret

res.insert(ret.begin(), 1))
